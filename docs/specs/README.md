# Specs

This directory contains design specifications and action plans for the
project: background and decision log ([mission.md](mission.md)), feature
designs, and implementation plans. Planned and in-progress work is tracked
as [GitHub Issues](https://github.com/borco/pyside6-scintilla/issues), not
in this directory. Use it to capture the *why* behind non-obvious
decisions before or while implementing them, not as a record of work
already visible in the code or git history.
